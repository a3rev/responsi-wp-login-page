The Responsi WP Login Page allows admin to edit the login page via the customizer menu 

== Description ==
Install and activate the Responsi WP Login Page and it adds WP Login Page menu to the Pages menu on the customizer.

== Features ==

* Create a custom design for your login page via the WordPress customizer
* WP Login page shows in the customizer when you access the WP Login Page Menu
* Customize the login page while you are logged in as admin


== Installation ==
Minimum Requirements
* WordPress 4.4
* PHP version 5.5 or greater
* MySQL version 5.5 or greater


== Usage ==
1. Install Responsi WP Login Page plugin from the Appearance > Add-on Plugins menu and activate it
2. Go to Pages > WP Login Page menu and start customizing the login page that shows in the preview
